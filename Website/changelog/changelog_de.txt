Changelog JBirthdays http://elmar-baumann.de/JBirthdays
=======================================================

0.4.12 2019-10-25
-----------------

* Tab "Über" zeigt benutzte Java- und Betriebssystem-Version an.
* Probleme mit neueren Java-Versionen als 1.8 behoben.

0.4.11 2018-01-28
-----------------

* Skalierung korrigiert für 4K-Bildschirme.

0.4.10 2017-10-02
-----------------

* Die Prüfung auf die neueste Version versucht nicht mehr, diese herunterzuladen.
  Statt dessen bietet sie an, die Download-Seite zu öffnen.

* 0.4.9 2017-07-17
----------------

* Ist die Bildschirmauflösung größer als Wide QHD QHD (2560 x 1440 Pixel),
  werden die Schriftarten um das 1,5-fache vergrößert. Ab Ultra High Definition
  "4K" (3840 x 2160 Pixel) werden die Schriftarten um das 2-fache vergrößert.

0.4.8 2014-12-10
----------------

* Beim Import von Daten aus einer JBirthdays-Datei wird vorher geprüft,
  ob diese gültig ist
* Es können nur Dateien mit der Endung ".xml" ausgewählt werden für den
  Import von Personen

0.4.7 2014-08-16
----------------

* Bugfix: Laden von Personen nach dem Speichern geschieht im gleichen Thread.
  Vorher war es möglich, dass nach dem Speichern nicht der richtige Zustand
  angezeigt wurde. Wurde beispielsweise eine neue Person hinzugefügt, erschien
  diese möglicherweise nicht in der Personenliste. Das gleiche war möglich
  nach dem Ändern von Personendaten oder Löschen von Personen.

0.4.6 2014-03-01
----------------

* In zukünftigen Geburtstagen wird das neue Alter angezeigt

0.4.5 2014-02-20
----------------

* Spaltenbreiten werden beim Beenden gespeichert

0.4.4 2014-02-01
----------------

* Bugfix: Wochentag für vergangene Geburtstagstermine

0.4.3 2014-02-01
----------------

* "Werkzeuge > Fehler melden" hinzugefügt
* Beenden-Schaltfläche hinzugefügt
* Bugfix: Wochentag des Geburtstags

0.4.2 2014-02-01
----------------

* Bugfix: Alter einer Person

0.4.1 2014-01-29
----------------
* Bugfix: Es war möglich, dass nach Start keine
  Personen angezeigt wurden

0.4.0 2014-01-16
----------------
* Geburtstage bestimmter Personen können von der
  Benachrichtigung ausgeschlossen werden
* Protokolldatei kann angezeigt werden
* Datenbank kann in andere Ordner verschoben werden
* Unter Windows wird angeboten, heruntergeladene neue
  Versionen von JBirthdays zu installieren

0.3.0 2014-01-14
----------------
* Datenbank kann ausgewählt werden
* Export und Import von Personen hinzugefügt zum Backup
  und Austausch mit anderen JBirthdays-Anwendungen

0.2.0 2014-01-13
----------------
* Auto-Update hinzugefügt
* An Personen, die Geburtstag haben, kann eine E-Mail
  gesendet werden

0.1.0 2014-01-12
----------------
Projekt gestartet
